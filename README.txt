Blog Me Do
==========

Prerequisites
=============
- An internet connection
- Maven 3.x
- JDK 7.0 and up

Arguments
=========
-Dserver.port - Sets the server's port. Default 8080

Goal - to build a web based blog system
Use cases:
#User can create a new blog post by providing: author's email address, blog title, and blog post content
#View last 10 blog titles (ordered by creation date)
#Clicking on a blog title will open a new page with all the blog content: author, title, and content
General:
#System should work either locally or on any cloud provider
#Code should include tests
#Code should be polished and elegant
